knowbox
=======

|build-status|

Installing ``knowbox`` is easy::

    pip install knowbox

Development
-----------

Increment the version number using ``bump2version``.::

    bump2version [major|minor|patch]

New releases are automatically deployed to PyPI via Travis CI.
